*DOWNLOAD*

Clone direclty from the WRF Github repository https://github.com/wrf-model the latest version of the model.
Create a folder call WRF-Chem. Go into it. From command line type:

   $  git clone https://github.com/wrf-model/WRF
   $  git clone https://github.com/wrf-model/WPS
   
*ENVIRONMENT TESTS AND INSTALLING LIBRARIES*
Follow instructions of the first tutorial of 1). More infos in the README in folder MACHINE_TESTS and LIBRARIES.

*WPS AND WRF-CHEM COMPILING*
Follow instructions of the second tutorial of 1). More infos in the README in folders WPS and WRF.
